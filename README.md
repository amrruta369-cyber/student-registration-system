# Student Registration System

## Description
This project is developed as part of an Internshala assignment.
It is a Student Registration System built using JavaScript DOM manipulation that allows users to register students through a form and view the data dynamically on the webpage.

## Features
- Add, edit, and delete student records
- Input validation
- Data persistence using Local Storage
- Fully responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How to Run the Project
1. Download or clone the repository
2. Open the project folder in any code editor
3. Open index.html in a web browser
